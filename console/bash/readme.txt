Before running this script for the first time, please edit it and modify 'hostname' and 'md5_password' with your own values.

If you don't know your own md5_pass please visit: 
https://www.duiadns.net/account/account-info.html

By default, this script runs for IPv4 updates (see the line: use_ip_version=4). You cand change this to "use_ip_version=6" if you want another script for IPv6 as well.
Tested on: linux, freebsd, tomato, openwrt, ddwrt, rasbian
