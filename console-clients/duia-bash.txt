This is a console version duia client and it was tested on many unix based OS
 versions like linux (many distributions), freeBSD, tomato, open-WRT & DD-WRT.

This script is using "wget". If you prefer "curl" instead, uncomment "curl" lines and comment (#) "wget" ones.

Before running this script for the first time, please edit it and modify 'hostname' and
'md5_password' with your own values
